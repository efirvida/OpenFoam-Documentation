# KinematicParcel.T 
## Class
Foam::KinematicParcel

## Description
Kinematic parcel class with rotational motion (as spherical
particles only) and one/two-way coupling with the continuous
phase.

Sub-models include:
- drag
- turbulent dispersion
- wall interactions

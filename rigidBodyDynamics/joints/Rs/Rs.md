# Rs 
## Class
Foam::RBD::joints::Rs

## Description
Spherical joint for rotation about the x/y/z-axes using a quaternion
(Euler parameters) to avoid gimble-lock.

Reference:
```
        Featherstone, R. (2008).
        Rigid body dynamics algorithms.
        Springer.
        Chapter 4.
```

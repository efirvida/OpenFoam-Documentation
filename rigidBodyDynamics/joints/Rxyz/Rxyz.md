# Rxyz 
## Class
Foam::RBD::joints::Rxyz

## Description
Spherical joint for rotation about the x/y/z-axes using Euler-angles
in the order x, y, z.

Reference:

        Featherstone, R. (2008).
        Rigid body dynamics algorithms.

# IOMRFZoneList 
## Class
Foam::IOMRFZoneList

## Description
List of MRF zones with IO functionality.  MRF zones are specified by a list
of dictionary entries, e.g.

```
zone1
{
        cellZone    rotor1;
        active      yes;
        ...
}

zone2
{
        cellZone    rotor2;
        active      yes;
        ...
}
```

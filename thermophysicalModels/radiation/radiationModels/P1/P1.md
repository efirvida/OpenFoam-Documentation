# P1 
## Class
Foam::radiation::P1

## Description
Works well for combustion applications where optical thickness, tau is
large, i.e. tau = a*L > 3 (L = distance between objects)

Assumes
     - all surfaces are diffuse
     - tends to over predict radiative fluxes from sources/sinks

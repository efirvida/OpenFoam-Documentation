# buoyancyForce 
## Class
Foam::fv::buoyancyForce

## Description
Calculates and applies the buoyancy force rho*g to the momentum equation
corresponding to the specified velocity field.

## Usage
Example usage:
```
buoyancyForceCoeffs
{
        fieldNames      (U);                    // Name of velocity field
}
```

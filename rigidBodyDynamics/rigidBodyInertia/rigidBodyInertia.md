# rigidBodyInertia 
## Class
Foam::rigidBodyInertia

## Description
This class represents the linear and angular inertia of a rigid body
by the mass, centre of mass and moment of inertia tensor about the
centre of mass.

Reference:
```
        Featherstone, R. (2008).
        Rigid body dynamics algorithms.
        Springer.
```

# PatchInjection.T 
## Class
Foam::PatchInjection

## Description
Patch injection.

User specifies:
      - Total mass to inject
      - Name of patch
      - Injection duration
      - Initial parcel velocity
      - Injection volume flow rate

Properties:
      - Parcel diameters obtained by distribution model
      - Parcels injected randomly across the patch

# coordinateRotation 
## Class
Foam::coordinateRotation

## Description
Abstract base class for coordinate rotation

```
coordinateRotation
{
        type        axesRotation
        e1          (1 0 0);
        e2          (0 1 0);
}
```

Types of coordinateRotation:
      -# axesRotation
      -# STARCDRotation
      -# cylindrical
      -# EulerCoordinateRotation

# KinematicCloud.T 
## Class
Foam::KinematicCloud

## Description
Templated base class for kinematic cloud

- cloud function objects

- particle forces, e.g.
      - buoyancy
      - drag
      - pressure gradient
      - ...

- sub-models:
      - dispersion model
      - injection model
      - patch interaction model
      - stochastic collision model
      - surface film model

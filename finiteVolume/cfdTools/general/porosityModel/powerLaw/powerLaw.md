# powerLaw 
## Class
Foam::powerLaw

## Description
Power law porosity model, given by:

        \f[
            S = - \rho C_0 |U|^{(C_1 - 1)} U
        \f]

where
\vartable
        C_0      | model linear coefficient
        C_1      | model exponent coefficient
\endvartable

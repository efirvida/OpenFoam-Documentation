## Class
Foam::ManualInjection

## Description
Manual injection.

User specifies:
      - Total mass to inject
      - Parcel positions in file \c positionsFile
      - Initial parcel velocity

Properties:
      - Parcel diameters obtained by distribution model
      - All parcels introduced at SOI
